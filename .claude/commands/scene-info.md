# Scene / Project Info

Show a quick summary of the current project and scene structure.

## Steps

1. List project files: `assets/projects/*.json` — read the active project
2. List scene files: `assets/scenes/*.json`
3. List scripts: `assets/scripts/*.lua`
4. List custom GLSL shaders in `GLSL/` that are NOT part of the built-in set
5. Show current `todo.md` known issues

Report: project name, active scenes, script count, custom shaders, open bugs.
